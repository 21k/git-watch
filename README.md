# git.watch

Full tutorial at: <http://git.watch>

### Setting up the hooks

#### GitHub

1. Go to "Settings" in your repository
2. Press "Webhooks & services"
3. Press "Add webhook"
4. In "Payload URL", write "http://git.watch/github/"
5. Press "Add webhook"
